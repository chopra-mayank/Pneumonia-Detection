Pneumonia Detection Using Chest X-Ray Images

This project leverages deep learning to detect pneumonia from chest X-ray images. It features a VGG16-based Convolutional Neural Network (CNN), a Flask web application for real-time predictions, and performance visualizations.

üîç Project Highlights

  • Model: VGG16-based CNN with transfer learning and dropout regularization.
  • Training: Fine-tuning with the last 4 convolutional layers unfrozen.
  • Visualization: Real-time training visualization (accuracy & loss).
  • Deployment: Flask web app for real-time predictions.

üìÅ Repository Structure

  • Training-Pneumonia.py: Script for training the CNN model.
  • Testing-Pneumonia.py: Script for evaluating the model.
  • Score.py: Utility for calculating performance metrics.
  • app.py: Flask application for deploying the model.
  • templates/ and static/: Directories for Flask's frontend components.
  • requirements.txt: List of Python dependencies.
